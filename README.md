## 要件定義

### サービス概要

島のお宿 Haruは、1日1組限定で伝統的な日本家屋を利用した宿泊体験を提供。
周辺には豊かな自然や瀬戸内国際芸術祭のアート作品があり、自然とアートが融合した特別な体験を楽しめる。

---

### 背景

現状の公式ウェブサイトは情報が不足しており、宿の魅力が十分に伝わらず、予約につながっていない状況。これを改善するため、新しいウェブサイトを構築する。  

公式ウェブサイト: [Haru Official Website](https://harushimayado.wixsite.com/haru)

---

### 目的

- **認知度向上と集客促進**  
  海外の旅行者をターゲットに、魅力的な情報を発信し、予約件数を増加させる。

- **情報提供の強化**  
  宿の特徴や周辺の観光スポットについて、充実したコンテンツを提供する。

---

### 現状の課題と解決策

#### (1) 情報不足

- **課題**  
  現在のウェブサイトでは、Haruの特徴や宿泊体験の魅力が十分に伝わっていない。

- **解決策**  
  1. **コンテンツの充実**  
     - 民宿Haruの特徴や豊島の観光情報を詳しく掲載する。  
     - ブログ機能を活用し、最新情報を定期的に発信する。
  2. **視覚的魅力の向上**  
     - 魅力的なデザインと宿泊施設の写真を多用し、印象を強化する。

---

#### (2) 外国人観光客向けの最適化不足

- **課題**  
  英語対応が不十分で、海外の観光客には情報が分かりづらい。

- **解決策**  
  1. **多言語対応サイト**  
     - 英語対応に加え、自動翻訳機能で多言語サポートを提供する。
  2. **ユーザビリティ向上**  
     - レスポンシブデザインを採用し、スマートフォンやタブレットでも快適に利用可能な環境を整える。

---

#### (3) 外部予約サイトへの依存

- **課題**  
  Airbnbなどの外部予約サイトへの依存により、予約管理が統一されていない。

- **解決策**  
  1. **独自の予約システムの導入**  
     - Haru専用の予約システムを構築し、直接予約を可能にする。  
  2. **予約管理機能の強化**  
     - カレンダー機能を追加し、予約状況を効率的に管理する。

---

### 差別化できる点

1. **伝統的な日本家屋**  
   - 豊かな歴史と文化を感じられる滞在を提供する。
2. **1棟貸し**  
   - プライバシーを重視し、広々とした空間を1組で利用可能とする。
3. **1日1組限定**  
   - 特別感のある宿泊体験を演出する。

---

### ターゲットユーザー

- **年齢層**: 25～50歳  
- **性別**: 男女問わず  
- **地域**: 欧米やヨーロッパからの外国人旅行者（全体の90%）  
- **興味・関心**: アート、自然、文化的な体験を求める層

## コマンド操作

1. **Dockerコンテナの起動**

    ```
    cd docker/haru
    docker-compose up -d
    ```
    
2. **ライブラリのインストール**

    ```bash
    //コンテナの中に入る
    docker-compose exec php-apache bash
    
    //ライブラリのインストール
    composer install
    ```
    
3. **Dockerコンテナの停止**

    ```
    docker-compose down
    ```
